Website Analytics Dashboard:
                               Website Analytics Dashboard is a Python-based interactive dashboard built using Streamlit to visualize key website performance metrics such as 
                               total visitors, page views, bounce rate, and session duration.

Tools Used:
            1.)Python (data handling and logic)
            2.)Pandas (data analysis)
            3.)Streamlit (dashboard UI)
            4.)Matplotlib & Plotly (visualizations)

Process Flow:
            1.)Load website analytics CSV data
            2.)Clean and process using Pandas
            3.)Compute KPIs and aggregate metrics
            4.)Visualize data using charts (line, bar, pie)
            5.)Display all insights in a clean dashboard layout

Output(Displayed in the Screenshot):
            1.)KPIs (Key Performance Indicator) at the top
            2.)Line chart (Visitors over Time)
            3.)Bar chart (Page Views by Country)
            4.)Pie charts (Device and Traffic Source)
            5.)Data preview at the bottom
