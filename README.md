Website Analytics Dashboard:
                               Website Analytics Dashboard is a Python-based interactive dashboard built using Streamlit to visualize key website performance metrics such as 
                               total visitors, page views, bounce rate, and session duration.

Tools Used:
            Python (data handling and logic)
            Pandas (data analysis)
            Streamlit (dashboard UI)
            Matplotlib & Plotly (visualizations)

Process Flow:
            Load website analytics CSV data
            Clean and process using Pandas
            Compute KPIs and aggregate metrics
            Visualize data using charts (line, bar, pie)
            Display all insights in a clean dashboard layout

Output(Displayed in the Screenshot):
            KPIs (Key Performance Indicator) at the top
            Line chart (Visitors over Time)
            Bar chart (Page Views by Country)
            Pie charts (Device and Traffic Source)
            Data preview at the bottom
